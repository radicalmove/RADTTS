    Step-by-step setup

    1. Open RADTTS in your browser and select your project.
    2. Click Set up helper app.
    3. Choose Windows installer or Mac installer and run it.
    4. If your browser blocks downloads, use Copy Mac setup command or Copy Linux setup command and run it in Terminal.
    5. Approve any security prompts once, then allow the helper app to run in the background.
    6. Return to RADTTS and click Refresh helper status to confirm it is online.

    Troubleshooting

    • If installer download does not start, copy and run the setup command manually.
    • If macOS blocks the app, go to System Settings > Privacy & Security and allow it, then run again.
    • If status stays offline, close and reopen the browser tab, then refresh helper status.
    • If command install fails, check Python is installed (`python3 --version`) and try again.
    • If helper count still does not increase, ask an admin to regenerate setup links and retry.
